## Proje Yapısı

Bu repository iki ana projeden oluşmaktadır:

### 1. quick-quiz/ - React JS Quiz Uygulaması
Basit, çok sorulu bir quiz uygulaması. Uygulama her seferinde bir soru gösterir, kullanıcının puanını takip eder ve sonunda bir sonuç ekranı gösterir.

**Özellikler:**
- Soruları gösterme
- Cevap seçeneklerini gösterme
- Puan takibi
- Quiz'i ilerletme
- Sonuçları gösterme

**Teknolojiler:**
- React
- useState Hook
- Bileşen Kompozisyonu
- Props
- Koşullu Render Etme
- Liste Render Etme (.map())

**Çalıştırma:**
```bash
cd quick-quiz
npm install
npm start
```

### 2. simple-gallery/ - React Native SimpleGallery Uygulaması
Tek ekranlı bir resim galerisi uygulaması. Küçük resim (thumbnail) grid'i gösterir ve kullanıcı bir küçük resme dokunduğunda, o resmin daha büyük bir versiyonunu açıklamasıyla birlikte gösterir.

**Özellikler:**
- Resim grid'i (FlatList ile)
- Seçili resmi gösterme
- Seçimi yönetme
- Boş durum (placeholder)
- Stil (StyleSheet.create())

**Teknolojiler:**
- React Native
- Expo
- useState Hook
- FlatList
- SafeAreaView, View, Text, Image, Pressable
- StyleSheet

**Çalıştırma:**
```bash
cd simple-gallery
npm install
npm start
```
